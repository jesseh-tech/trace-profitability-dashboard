📥 Step-by-Step: Loading Data Each Month
1

Export from Paycom

Log into Paycom → Reports → Labor Allocation by Job. Set the date range to the month you want. Export as .xlsx. This file covers both Trace and CC&E — one export handles both.

2

Export from QuickBooks — Trace

In QuickBooks Online for Trace LLC → Reports → Sales by Location (or Sales Transactions). Filter by the same date range. Export as .xlsx or .csv.

3

Export from QuickBooks — CC&E

In QuickBooks Online for CC&E → Reports → Project Costs Detail. Same date range. Export as .xlsx or .csv.

4

Upload all three files

Drag each file into its matching drop zone in the Data Import section below. The dashboard processes everything automatically — no formatting required.

5

Check for unmatched codes

If a yellow warning appears, some labor hours couldn't be linked to a job. Use the Job Code Map section below to download the template, fill in the missing links, and re-upload. This is optional but improves accuracy.

📊 Reading the Dashboard
✓

KPI Cards (top row)

Shows total revenue, labor cost, all-in costs, gross profit, and median job margin for the selected company tab. Switch between Trace, CC&E, and Combined using the tabs.

✓

Job Table

Every row is a job. Click any row to expand the full cost breakdown — labor, materials, overhead, and employee-level detail from Paycom. Use the filters to sort by margin, revenue, or find a specific job.

✓

Margin color codes

Green ≥ 20% — healthy. Yellow 5–20% — watch it. Red < 5% — needs attention.

✓

Export Report

Click the Export Report button in the top right to download the current view as a .csv file. Open in Excel for further analysis or to share with your accountant.

📂 Step 1 — Data Import

Upload one combined Paycom export + separate QuickBooks exports per company. Accepts .xlsx, .xls, or .csv.

Paycom: Uses Distributed Job Location Code as the job match key.  |  Trace QBO: Sales Transactions report — matches on the Location field (Invoice & Estimate rows only; Payments excluded).  |  CC&E QBO: Project Costs Detail report — costs parsed per project; project number extracted from end of project name (e.g. Metro Center Light Poles - 1007 → 1007).
✅ Job Code Map built-in: 345 Trace QBO locations are pre-mapped to Paycom codes. All 302 existing codes are mapped, plus 43 new codes (1258–1300) assigned for jobs with no prior Paycom code. CC&E Paycom codes (e.g. 1007) match QBO project names automatically. You can upload a custom CSV override in Step 2 below.

To fix this: Download the template below, open it in Excel, fill in the QBO Location and Paycom Code columns for any new jobs, save as .csv, and upload it here. You only need to do this when new jobs appear.
